PCGEN is a character generator for the D20 roleplaying system; the main class is in the {@link pcgen.gui} package.

The program is object oriented, with the model (pcgen.core) partially separated from the view/controller UI (pcgen.gui). The start up main class for the application is pcgen.gui.pcGenGUI, which creates the main model object Globals and the main view object PCGen_Frame1.
