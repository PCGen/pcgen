Provides for core PCGen functions, mostly related to D&D 3e features.

This is where the business logic goes. This is the main engine that makes things work.

The most important class in here is PObject, many classes are subclassed from this (like PCClass, Race, Feat, Spell, etc.).

Another important class is PlayerCharacter.java which is where the playercharacters are constructed and manipulated.

Globals.java contains the lists of objects loaded when a user loads their desired sources.
