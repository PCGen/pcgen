PCGen

PCGen is a character generator for role-playing games. It is currently aimed at supporting as many RPGs as is humanly possible. The project's current focus is on OGL games (OGL is Wizards of the Coast's Open Gaming License). PCGen runs on Windows, Mac OS X and Unix/Linux using Java 1.7 (or later).

PCGen is a Java based open source project. All datafiles are ASCII so they can be modified by users.
