What's new for v6.6.00RC1

PCGen v6.6.00RC1 is a release candidate from our development branch to allow experimentation and testing of our latest work before a final production release. Please give it a shot and let us know how it works for you. Some notable changes in this release are listed below.

Notable Changes

  • Code stability fixes and improvement
  • Monk Flurry of Blows basic support for Outputsheet
  • Further stability and bug fixes

New Sources

The following new source was added in 6.6.00RC1

  • Pathfinder Player Companion: Weapon Master's Handbook

For full details of the new changes in this release, see the Change Log .

Andrew Maitland for the PCGEN Team -- 28 December, 2015.

Installation

Prerequisites

Java 7 or Java 8. PCGEN 6.6.00RC1 is not compatible with Java 6 (or earlier) and it is NOT SUPPORTED. To get the latest version of Java, go to java.com. PCGen has been tested with both Java 7 (including OpenJDK) and Java 8.

Instructions

Please see the Installation Documentation for information about installing PCGen.

Read the doc!

Please make sure you read the docs to familiarize yourself with the changes. You can access it from docs/index.html located in the PCGen directory.

Download PCGen

  • pcgen60600RC1_full.zip:
    everything including GMGen and PDF
  • pcgen60600RC1_win_install.exe:
    the full package bundle in an easy to use installer for Windows

Custom Data Sets

If you have custom data created using either the old LST Editor, or the my_dataset example files, it is very important that you read the Upgrading PCGen and Upgrading home brews before installing your new version of PCGen.

Want to Help?

PCGen is an open source project. Anyone who wants to contribute whether it is Java programming, data set programming, documentation, helping new users, general cat herding, massage, walking the dogs, etc. is very welcome.

If you want to help drop a mail to PR@pcgen.org or help@pcgen.org, or simply post a message with the title "I Want To Help" in one of the mailing lists.
