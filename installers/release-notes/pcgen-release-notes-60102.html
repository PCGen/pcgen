What's new for v6.1.2

PCGen v6.1.2 is the an alpha release from our development branch to allow experimentation and testing of our latest work. Please give it a shot and let us know how it works for you. Some notable changes in this release are listed below.

Notable Changes

  • Data converter has been streamlined
  • Performance improvements

New Sources

The following new sources were added in 6.1.2

  • [Paizo] Player Companion: Dragon Empires Primer
  • [Paizo] Halflings of Golarion

For full details of the new changes in this release, see the Change Log .

James Dempsey for the PCGEN Team -- 20 May, 2013.

Installation

Prerequisites

Java 6 or Java 7. PCGEN 6.1.2 is not compatible with Java 1.5.x and it is NOT SUPPORTED. To get the latest version of Java, go to java.com. PCGen has been tested with both Java 6 (including OpenJDK) and Java 7.

Instructions

Please see the Installation Documentation for information about installing PCGen.

Read the doc!

Please make sure you read the docs to familiarize yourself with the changes. You can access it from docs/index.html located in the PCGen directory.

Download PCGen

  • pcgen60101_full.zip:
    everything including GMGen and PDF
  • pcgen60101_win_install.exe:
    the full package bundle in an easy to use installer for Windows
  • pcgen60101_mac_install.dmg:
    the full package bundle in an easy to use installer for Mac (distributed as a disk image)

Character Changes in 6.1.2

Pathfinder barbarian rage powers for all Core Powers will need to be reselected with this release. When you load your character the existing core rage powers will be listed as unable to be found. Please note these down and load your character as normal. Once loaded a to-do entry will show you have rage powers to select. Click on the link and you can reselect the rage powers.

This change was necessary to complete support for barbarian archetypes in Ultimate Combat.

Custom Data Sets

If you have custom data created using either the old LST Editor, or the my_dataset example files, it is very important that you read the Upgrading PCGen and Upgrading home brews before installing your new version of PCGen.

Want to Help?

PCGen is an open source project. Anyone who wants to contribute whether it is Java programming, data set programming, documentation, helping new users, general cat herding, massage, walking the dogs, etc. is very welcome.

If you want to help drop a mail to PR@pcgen.org or Content@pcgen.org, or simply post a message with the title "I Want To Help" in one of the mailing lists.
