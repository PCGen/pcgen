What's new for v5.9.7 Beta

Well, this is it. The first Beta release of the 5.9.x branch. If all things go well, you should see one more Beta release and the first Release Candidate should be around April 8.

There was a massive reorganization of the SRD, RSRD and MSRD directories. Please see the following section for some very important information about this.

Important note: This is an Beta release. We make this release only to make it more easy for the users to test the development version. Use it at your own peril. For users that are looking for stability, please use the latest stable version.

Éric "Space Monkey" Beaudoin for the PCGEN Team -- March 27, 2006.

SRD, RSRD and MSRD Directory reorganization

In this release we have reorganized the core datasets directories. Essentially we have moved the files loaded by each Partial set into it's own directory. We have done this to simplify maintenance and to stream line their use, since each is now a self-contained dataset you can be confident that no file is double loaded when you mix and match Partial sets. You can also load any combination of sets and know that only the basics set is required (with the exception of the Epic Psionics set which also requires the Psionics set).

Impact: If you have created custom datasets which use relative links to load files from the core datasets these links will be broken. You will need to check out the new structure to see how it need to be fixed. Similarly if you have created a custom pcc with the Customize button in the Sources Tab you will need to manually edit it to correct the paths or simply recreate it.

For the SRD and RSRD this change is not expected to break any previously created characters since the dataset names were not changed.

Changes to the MSRD on the other hand are going deeper as they involve a major overhaul of the PCC Type and Campaign tags. Where the complete set previously only loaded the Basics and Menaces sections of the MSRD, there is now a much more modular structure that will help to better cope with the multitude of genres that can be covered by the Modern game.

The new setup now shows under Complete: MSRD Arcana, MSRD Basics, MSRD Future, and MSRD Menaces. These sets will each load a complete section, as those appear as partially downloadable sections of the MSRD on the WotC webpage. To exactly do that, the section previously called "Core" has been renamed to "Basics" as that is what the MSRD uses. Of course there are still the optional partial sets for users that want to load only certain components, like loading the Basics without FX.

But all those changes to the MSRD come at a price. Pre-existing characters will most certainly be broken and are unlikely to still work with this new version. To work around this you can try the following steps:

  • 1st-make a copy of your character's .PCG file, in case you might mess up with editing the original;
  • 2nd-open the original .PCG file in a text-editor;
  • 3rd-delete the CAMPAIGN:xxx tags in the file (i.e. CAMPAIGN:Modern);
  • 4th-save the file again;
  • 5th-start PCGen;
  • 6th-select and load all sources that your character has previously taken choices from;
  • 7th-load the character;
  • 8th-if the character now loaded successfully, save it in the new format. That will store which sources were loaded with it, so you can treat that file as normal in the future.

This procedure should usually work.

Installation

Prerequisites

Java 1.4.x or Java 5. PCGEN 5.9.7 is not compatible with Java 1.3.x and it is NOT SUPPORTED. To get the latest version of Java, go to java.com.

Windows Users

We recommend that you use the Windows installer.

Mac Users

We recommend that you use the OS X DMG Mac Installer, it is now working with OS X 10.4 (Tiger). This has been a problem in the past due to a change on Apple’s part in how the menu bar was handled with java 1.4 on Tiger, a packaged java app set to use the main window instead of the application window caused the program to crash at startup. At this point in time the solution is to set the menu bar to appear in application menu.

Also new in this version is the configuration of the program, all the necessary files are now included with the program package. You can still access the files by right clicking the program and choosing Show Package Contents. See the notes included in the installer DMG for all the details.

Others (or advanced) users

Pick the package with the features you want and unzip it. It is highly recommended that you use a new directory for every new release. Once the package is unzipped, see docs\index.html for the full documentation.

Alpha Data Sets

The Alpha directory contains data sets that were not judged to be "production" grade. They vary from "better then nothing" to "nearly ready for production". Since this is an Alpha release, the alpha folder is include.

Read the doc!

Please make sure you read the docs to familiarize yourself with the changes. You can access it from docs/index.html located in the PCGen directory.
