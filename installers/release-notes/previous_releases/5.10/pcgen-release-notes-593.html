What's new for v5.9.3 Alpha

I lot of things moved from core application to plugins and it might be just my imagination but the initial load seams a lot faster then 5.9.2 or 5.8.0 :-). We now have a dice roller and some new data sets.

Important note: This is an Alpha release. We make this release only to make it more easy for the users to test the development version. Use it at your own peril. For users that are looking for stability, please use the latest stable version.

Éric "Space Monkey" Beaudoin for the PCGEN Team -- September 18, 2005.

Installation

Prerequisites

Java 1.4.x or Java 5. PCGEN 5.9.3 is not compatible with Java 1.3.x and it is NOT SUPPORTED. To get the latest version of Java, go to java.com.

Windows Users

We recommend that you use the Windows installer.

Mac Users

Use the installation DMG file unless you are using OS X Tiger (there is a bug with the Mac executable when running on a Tiger machine).

Others (or advanced) users

Pick the package with the features you want and unzip it. It is highly recommended to use a new directory with every new release. Once the package is unzipped, see docs\index.html for the full documentation.

Alpha Data Sets

The Alpha directory contains data sets that were not judged to be "production" grade. They vary from "better then nothing" to "nearly ready for production". Since this is an Alpha release, the alpha folder is include.

Read the doc!

Please make sure you read the docs to familiarize yourself with the changes. You can access it from docs/index.html located in the PCGen directory.

  • pcgen593_full.zip (PGP Signature):
    everything including GMGen and PDF
  • pcgen593_partial.zip (PGP Signature):
    everything except GMGen and PDF
  • pcgen593_win_install.exe (PGP Signature):
    the full package bundle with Ed's nifty installer for Windows
  • pcgen593_mac_install.dmg (PGP Signature):
    the full package bundle with Connor's nifty installer for Mac (distributed as a disk image)
    Warning: do not use the Mac installer if you are running the latest Mac OS (Tiger)

Want to Help?

PCGen is an open source project. Anyone who wants to contribute whether it is Java programming, data set programming, documentation, helping new users, general cat herding, massage, walking the dogs, etc. is very welcome.

If you want to help drop a mail to Paul W. King (paulking.rhochi@verizon.net) or Karianna (karianna@clear.net.nz), or simply post a message with the title "I Want To Help" in one of the mailing lists (and may the gods have mercy on your soul :-).
