What's new for v6.5.2

PCGen v6.5.2 is the an alpha release from our development branch to allow experimentation and testing of our latest work. Please give it a shot and let us know how it works for you. Some notable changes in this release are listed below.

Notable Changes

  • Following game systems removed from main distribution: 3e, Deadlands, Gaslight (1st edition, replaced by 2nd edition), Legends of Excalibur, Sidewinder, Spycraft, Xcrawl
  • Many small fixes

New Sources

The following new sources were added in 6.5.2

  • [Paizo] Blood of the Moon
  • [Paizo] Campaign Setting: Distant Worlds
  • [Paizo] Pathfinder Adventure Path: Iron Gods Player's Guide
  • [Paizo] Pathfinder Campaign Setting: Inner Sea Combat
  • [Paizo] Pathfinder Campaign Setting: Technology Guide
  • [Paizo] Pathfinder Companion: Taldor, Echoes of Glory
  • [Paizo] Player Companion: People of the Stars
  • [Super Genius Games] Advanced Options: Inquisitors' Judgments

For full details of the new changes in this release, see the Change Log .

James Dempsey for the PCGEN Team -- 26 May, 2015.

Installation

Prerequisites

Java 7 or Java 8. PCGEN 6.5.2 is not compatible with Java 6 (or earlier) and it is NOT SUPPORTED. To get the latest version of Java, go to java.com. PCGen has been tested with both Java 7 (including OpenJDK) and Java 8.

Instructions

Please see the Installation Documentation for information about installing PCGen.

Read the doc!

Please make sure you read the docs to familiarize yourself with the changes. You can access it from docs/index.html located in the PCGen directory.

Download PCGen

  • pcgen60502_full.zip:
    everything including GMGen and PDF
  • pcgen60502_win_install.exe:
    the full package bundle in an easy to use installer for Windows
  • pcgen60502_mac_install.dmg:
    the full package bundle in an easy to use installer for Mac (distributed as a disk image)

Custom Data Sets

If you have custom data created using either the old LST Editor, or the my_dataset example files, it is very important that you read the Upgrading PCGen and Upgrading home brews before installing your new version of PCGen.

Want to Help?

PCGen is an open source project. Anyone who wants to contribute whether it is Java programming, data set programming, documentation, helping new users, general cat herding, massage, walking the dogs, etc. is very welcome.

If you want to help drop a mail to PR@pcgen.org or Content@pcgen.org, or simply post a message with the title "I Want To Help" in one of the mailing lists.
