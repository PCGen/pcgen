What's new for v6.07.10

PCGen v6.07.10 is released as a BETA from our development branch. It reflect new features intended for the next production release and now a switch to squash any bugs from new features. Please give it a shot and let us know how it works for you. Some notable changes in the 6.07.10 release are listed below.

Notable Changes

  • Data bugs fixed
  • Starfinder Preview Sheet Fixed
  • New Formula System Stabilized

For full details of the new changes in this release, see the Change Log .

Andrew Maitland for the PCGEN Team -- 2018-11-11.

Installation

Prerequisites

Java 9, 10 or 11. PCGEN 6.07.10 is not compatible with Java 8 (or earlier) and it is NOT SUPPORTED. To get the latest version of Java, go to AdoptOpenJDK. PCGen has been tested with Java 9, 10 & 11 (including OpenJDK).

Instructions

Please see the Installation Documentation for information about installing PCGen.

Read the doc!

Please make sure you read the docs to familiarize yourself with the changes. You can access it from docs/index.html located in the PCGen directory.

Custom Data Sets

If you have custom data created using either the old LST Editor, or the my_dataset example files, it is very important that you read the Upgrading PCGen and Upgrading home brews before installing your new version of PCGen.

Thanks

Thanks to the volunteers who contributed to the PCGen 6.07.10 release! Without volunteers, no releases would be possible.

Thanks also to the many people who gave us feedback, reported bugs, contributed to the lists and helped other PCGen users out!

Want to Help?

PCGen is an open source project. Anyone who wants to contribute whether it is Java programming, data set programming, documentation, helping new users, general cat herding, massage, walking the dogs, etc. is very welcome.

If you want to help drop a mail to PR@pcgen.org or help@pcgen.org, or simply post a message with the title "I Want To Help" in one of the mailing lists.
