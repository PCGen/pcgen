What's new for v5.17.9

PCGen v5.17.9 is an alpha release from our development branch to allow experimentation and testing of our latest work. Please give it a shot and let us know how it works for you. Some notable changes in this release are listed below.

  • New Fantasy Grounds output sheet
  • PCGen now requires Java 1.6

New Sources

The following new sources were added in 5.17.9

  • [Dreamscarred Press] Psionics Expanded - Find the Mark
  • [Dreamscarred Press] Psionics Expanded - Mind over Body
  • [Dreamscarred Press] Psionics Unleashed
  • [Paizo] Pathfinder Bestiary 2
  • [Paizo] Pathfinder Jade Regents' Players Guide

We are also working away on the next big version although much of that work is currently behind the scenes. We welcome your input, just comment on the wiki pages or post to the PCGen mailing list .

For full details of the new changes in this release, see the Change Log .

James Dempsey for the PCGEN Team -- 16 November, 2011.

GMGen Known Issue

There is a known issue with GMGen when used on a Macintosh. In the Initiative Tab when running a combat when you select any character other than the one whose turn it is and click the Damage or Heal buttons the character whose turn it is becomes selected and the damage is applied to him instead of the character which had been selected.

A solution has not yet been found but fortunately there is a simple work-around. The problem will only manifest when using the System Look and Feel, you can change the Look and Feel by going to Preferences/Appearance/Look and Feel and changing it to Java before activating GMGen and then it will work as it should.

Installation

Prerequisites

Java 6. PCGEN 5.17.9 is not compatible with Java 1.5.x and it is NOT SUPPORTED. To get the latest version of Java, go to java.com. Note also that PCGen does not work with Java 7 at this time, however this is being addressed as part of the work on the new user interface.

Important Warning

If you have custom data created using either the LST Editor, or the my_dataset example files, it is very important that you read the Upgrading PCGen and Upgrading home brews installing your new version of PCGen.

Custom Data Sets

PCGen provides two simple ways of adding your own data, these are the inbuilt LST Editors and the starter data set my_dataset. If you have used these, please see the Upgrading PCGen documentation when upgrading PCGen.

Want to Help?

PCGen is an open source project. Anyone who wants to contribute whether it is Java programming, data set programming, documentation, helping new users, general cat herding, massage, walking the dogs, etc. is very welcome.

If you want to help drop a mail to PR@pcgen.org or Content@pcgen.or, or simply post a message with the title "I Want To Help" in one of the mailing lists (and may the gods have mercy on your soul :-).
