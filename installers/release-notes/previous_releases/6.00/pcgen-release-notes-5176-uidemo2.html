What's new for UI Demo 2

This is our second demo of the new user interface. We've taken on board much of the excellent feedback from the beta group. Some changes could not be squeezed into this release, but we're still planning on tackling them soon.

So please give this new version a try and continue to provide feedback on the user interface is working.

Here's what we've changed:

  • Fixed NPE in preferences dialog
  • Set size of columns in summary tab statistics section.
  • Reduce log output
  • Add support for adding levels when race is set
  • Adjusted initial frame size and fixed info pane layout issues
  • Add tooltip to method button
  • Ensure stat mods are updated when rolling stats.
  • Hide internal abilities from the UI
  • Focus traveral on SummaryTab now operates correctly
  • Added space between SummaryTab components
  • UI update
  • Reimplemented memory display in debugging console
  • Various UI fixes
  • Interim fix for loading weapons in equipment sets
  • Enforce ordering of todos
  • Add preliminary hyperlink support (dialog displayed for now)
  • Add todos for skills
  • Tidy up terminology in todo messages
  • Move languages to be after race in todo
  • Update skill sheet each time skills are changed
  • Allow summary info to be copied.
  • [race & class] The top part of the dropdown should have the standard races/classes first, then run things alphabetically.
  • Equipping tab - cater for slots such as rings which can occur multiple times.
  • Fix bug: CODE-555 - Could not save character - should ask for a new location
  • Add a new summary sheet for Pathfinder

Some things we still intend to look at are:

  • Should not be able to load sources without their pre-req's, like the APG
  • FEATS TAB - hide entries that dont pertain to character
  • [ability scores] Stack the "Editable Score" title to shrink the row.
  • No stat bump chooser
  • Ability Scores table ordering
  • Race combo box - keyboard nav shouldn't select until drop-down closed
  • No wizard speciality chooser
  • No general chooser (e.g. feats selections)

James Dempsey for the PCGEN Team -- 26 April, 2011.
