Toolbar Quick Icons

The Toolbar is a quick and efficient way to do many simple tasks within PCGen without the need to click on any drop down menus.

The Toolbar Quick Icons are located below the menu names near the top left of the main PCGen Window.

The functions in order from left to right are as follows:

Click on the New icon to create a new character

To open a file from a folder select the Folder icon

Close the current character

Save the current character

Print the current character

Export the current character

To pen the preferences window
