Requirements for PCGen

  • All versions of PCGen need Java Runtime Environment (JRE) which is available for all common OS's, including Windows, macOS, UNIX and Linux.
  • PCGen needs to run under Java 11 or above.
  • If you wish to view PDF output sheets, you'll need a PDF viewer.
  • Minimum RAM is 256MB (recommended 1-2GB)
  • Minimum CPU is 1GHz (recommended 2Ghz+)
  • 200MB of free HD space
