Using IntelliJ IDEA

This guide will help you get up to speed on setting up your local development environment with IntelliJ IDEA Community or Ultimate Edition.

First steps

The IDE IntelliJ IDEA is the commercial project developed by JetBrains. The IDE has two versions: Ultimate (commercial) and Community Edition (free). It can be downloaded from their official link. IDEA can open the PCGen as a Gradle project, or as a file-based project (an .ipr file).

The Gradle project is a wrapper around gradle.build file. You have to open the PCGen folder and then add a Gradle run task to build and start the project.

The .ipr file can be generated by a Gradle task:

gradle idea

The generated .ipr file will be mentioned in the console output:

> Task :idea
Generated IDEA project at file:///Users/your_user/projects/pcgen/pcgen.ipr

If you are experimenting with the .ipr file generation (e.g., correcting build.gradle file). You can regenerate the .ipr and other files using the task cleanIdea:

gradle cleanIdea
or 
gradle cleanIdea && gradle idea

p.s. if the tasks are combined together (gradle cleanIdea idea), the generated project file might still have changes from previous attempts.

After you generate the .ipr file, you can open it as a project and then use the Application run/debug configuration.
