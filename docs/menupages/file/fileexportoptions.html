Export Options

The Export option allows the user to export a character from PCGen to another format to be used or viewed by other applications.
It also allows you to export your character to a human readable format that can be printed out and used in your game without a computer.
When the user selects the Export option it will prompt for one of three additional selections: Standard, to PDF or to Text.
These selections allow the user to export the program to a standard format: XML or html, to a PDF file to be viewed with Adobe Acrobat reader or a plain Text file.
