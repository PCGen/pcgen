Operators

ADD (Adds to value)

SUBTRACT (Decreases the value)

FLOOR (returns lowest whole number - rounds down)

CEIL (returns highest whole number - rounds up)

MIN (returns the lowest value of two vars (a,b))

MAX (returns the highest value of two vars (a,b))

SET (Sets the value of the var to the specified value)

Priority

is an order of operation tag that may be added to MODIFY and MODIFYOTHER. DEFAULT is 0, the order is lowest to highest, with highest being last to run.

MODIFY and MODIFYOTHER

*** New 7.00.00 (Dev 6.07.01)

Tag Name: MODIFY:a|b|c|d

Variables Used (a): Text (Object)

Variables Used (b): Text (Operator)

Variables Used (c): Text/Number/Formula (Value)

Variables Used (d): Text (PRIORITY)

What it Does:

MODIFIES the object by the formula set forth.

Example:

Example:

MODIFY:MonkeyBreathAttack|ADD|1|PRIORITY=10

The value of MonkeyBreathAttack is increased by 1

*** New 7.00.00 (Dev 6.07.01)

Tag Name: MODIFYOTHER:a|b|c|d|e|f

Variables Used (a): Text (Scope)

Variables Used (b): Text (Object or GROUP)

Variables Used (c): Text (Variable Name)

Variables Used (d): Text (Operator)

Variables Used (e): Text/Number/Formula (Value)

Variables Used (f): Text (PRIORITY - Optional)

What it Does:

MODIFIES the object by the formula set forth.

Example:

MODIFYOTHER:EQUIPMENT|GROUP=Armor|Eq_MaxDex|SET|1000|PRIORITY=1

The value of Eq_MaxDex is set to 1000
