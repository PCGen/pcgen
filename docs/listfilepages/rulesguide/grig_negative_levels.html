Equipment Granting Negative Levels

Rule Description: Items can now grant "Negative Levels".

PCGen Version: 5.8.x to current.

File(s) Covered: equipment.lst and feat.lst

Tags used:

TYPE, VISIBLE, SA, DEFINE, BONUS:CHECKS, BONUS:COMBAT, BONUS:SKILL VFEAT, BONUS:VAR PRExxx

Implementation Discussion

In this section we will discuss only those lst tag elements specific to this rules implementation. For more general information on the associated tags, please visit the LST Tag section of the PCGen Documentation.

Implementing the ability for an item to grant 'Negative Levels" requires two specific LST entries. These are a feat.lst entry titled 'Negative Levels" and a specifc application of this feat in the appropriate item entry in the equipment.lst file.

NOTE: This game rule has been icorporated in the SRD, RSRD, MSRD, and several other datasets currently in distribution.

Feat File Entry

In the specific feat entry in the feat.lst file we must first create a new feat called "Negative Levels".

Equipment File Entry

In the specific equipment entry in the equip.lst file we must first apply the new "Negative Levels" by using the VFEAT tag:

Tag Name: VFEAT:Negative Levels

What it does:

Applies the Negative Levels feat and initializes the NegLevels variable.

Example:

VFEAT:Negative Levels

Use "Negative Levels" virtual feat

Second, create the item as normal and append to it a special BONUS tag.

Tag Name: BONUS:VAR|NegLevels|x|PREy

Variables Used (x): Number (number of negative levels.)

Variables Used (y): Text (PRE condition for bestowing negative levels)

What it does:

  • "NegLevels" is a special predefined variable made specifically for this purpose.
  • Use '2', not -2, to bestow 2 negative levels.
  • If the NegLevels Variable is 0 (no negative levels applied), then nothing will show, and it will be as if the Negative Levels feat was not even there. :)

Example:

BONUS:VAR|NegLevels|1|PREALIGN:0,1,3,4,6,7

Demon Armor bestows 1 negative level to all non-evil characters.

LST File Entry Examples

These examples come from the RSRD.

Feat File: "Negative Levels" feat

Negative Levels
TYPE:Special
VISIBLE:NO
SA:% negative level(s) (-% effective level(s) and loses access to % spell(s) from the highest spell level castable)|NegLevels|NegLevels|NegLevels
DEFINE:NegLevels|0
BONUS:CHECKS|Fortitude,Reflex,Will|-1*(NegLevels)
BONUS:COMBAT|TOHIT|-1*(NegLevels)
BONUS:SKILL|TYPE=Strength,TYPE=Dexterity,TYPE=Constitution,TYPE=Intelligence,TYPE=Wisdom,TYPE=Charisma|-1*(NegLevels)
