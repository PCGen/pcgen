What's new for v5.9.1 Alpha

Here we go again :-). This is the first Alpha release that will eventualy lead us to the next stable release which will be either 5.10.0 or 6.0.0.

Important note: This is an Alpha release. We make this release only to make it more easy for the users to test the development version. Use it at your own peril. For users that are looking for stability, please use the latest stable version.

Éric "Space Monkey" Beaudoin for the PCGEN Team -- July 11, 2005.

Installation

Prerequisites

Java 1.4.x or Java 5. PCGEN 5.9.1 is not compatible with Java 1.3.x and it is NOT SUPPORTED. To get the latest version of Java, go to java.com.

Windows Users

We recommend that you use the Windows installer.

Mac Users

Use the installation DMG file unless you are using OS X Tiger (there is a bug with the Mac executable when running on a Tiger machine).

Others (or advanced) users

Pick the package with the features you want and unzip it. It is highly recommended to use a new directory with every new release. Once the package is unzipped, see docs\index.html for the full documentation.

Alpha Data Sets

The Alpha directory contains data sets that were not judged to be "production" grade. They vary from "better then nothing" to "nearly ready for production". Since this is an Alpha release, the alpha folder is include.

Read the doc!

Please make sure you read the docs to familiarize yourself with the changes. You can access it from docs/index.html located in the PCGen directory.
