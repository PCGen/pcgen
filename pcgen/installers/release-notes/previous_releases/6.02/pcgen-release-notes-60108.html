Release Notes for PCGen v6.1.8

PCGen is a FREE Open Source character generator and maintenance program for role-playing games.

PCGen currently supports the d20 RPG system and includes data sets based upon gaming material from Wizards of the Coast, Paizo Publishing and dozens of other publishers. The project's current focus is on gaming material released under Wizards of the Coast's Open Gaming License (OGL) and sources supporting the Pathfinder RPG by Paizo, but the PCGen team also works with publishers to get permission to include limited Closed Content as well. PCGen runs on Windows, Mac OS X and Unix/Linux using Java

What's new for v6.1.8

PCGen v6.1.8 is an alpha release from our development branch to allow experimentation and testing of our latest work. Please give it a shot and let us know how it works for you. Some notable changes in this release are listed below.

Notable Changes

  • Pathfinder Overhaul to standardize naming and layout
  • Improved Company/Format/Setting/Name display for Pathfinder in source selection
  • Ultimate Equipment and Bestiary 3 included in the 'Pathfinder RPG Core Supplements' source
  • Bestiary 3 included in the 'Pathfinder RPG for Game Masters' source
  • Waiting animation on character sheet generation in the tab
  • Conditional source loading using PRECAMPAIGN and PRERULE

New Sources

The following new sources were added in 6.1.8

  • [Paizo] Pathfinder Player Compendium: Faiths of Balance
  • [Paizo] Pathfinder Player Compendium: Faiths of Corruption

For full details of the new changes in this release, see the Change Log .

Java 6 or Java 7. PCGEN 6.1.8 is not compatible with Java 1.5.x and it is NOT SUPPORTED. To get the latest version of Java, go to java.com. PCGen has been tested with both Java 6 (including OpenJDK) and Java 7.

Changes Affecting Homebrew Pathfinder Data in 6.1.8

With the huge number of sources we are supporting for Pathfinder, we had to do some further reorganizing for better maintainability and closer alignment to the publishers abbreviations and terms. The changes WILL impact our homebrew community, but they are limited to the Pathfinder sets only.

The main changes were

  • Renamed campaigns to match the publisher's names and remove redundant info. e.g. "Paizo - Pathfinder Roleplaying Game Ultimate Combat" became "Ultimate Combat". See system/gameModes/Pathfinder_RPG/migration.lst for a comprehensive list of renames.
  • Removed the Paizo - Pathfinder Roleplaying Game Character Traits Web Enhancement source (APG can be used for character traits).
  • Merged multiple files in a source of the same type into a single or limited number of files (e.g. all core rulebook equipmod files into cr_equipmods.lst
  • Renamed source folders to match publisher names
  • Renamed the pathfinder/paizo/pathfinder folder to pathfinder/paizo/roleplaying_game . This is the folder containing the main rule books.
  • Moved the main source definitions (e.g. pathfinder_for_gms.pcc) to be in the pathfinder/paizo folder.

The use of the migration tools means that this change should not impact characters using the sources included in PCGen. Instead the characters should be updated to the new source names on load.
