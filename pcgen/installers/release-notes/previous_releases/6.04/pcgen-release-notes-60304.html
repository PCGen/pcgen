What's new for v6.3.4

PCGen v6.3.4 is our first beta release for the 6.4 cycle! This version has all the features we want to put out in our next production release.

From here the focus changes to fixing any issues you report and generally getting PCGen polished up ready for production. So please grab a copy, load up your favourite characters and let us know how it works for you. In particular please check your characters output correctly in preview, html and PDF. We've tested lots of combinations but there may still be a few cases where there are output problems.

Some notable changes in this release are listed below.

Notable Changes

  • Lots of fixes to output
  • Many data fixes
  • Update Guide to PSOP to recently released 5.0 version
  • Improved error handling when exporting characters
  • Converter run on all sets

New Sources

The following new sources were added in 6.3.4

  • [Alluria Publishing] Remarkable Races: Compendium of Unusual PC Races, Pathway to Adventure Edition
  • [Paizo] Reign of Winter Players Guide
  • [Paizo] Rise of the Runelords (Anniversary Edition)
  • [Paizo] Skull & Shackles AP55 - The Wormwood Mutiny
  • [WotC] Unearthed Arcana

For full details of the new changes in this release, see the Change Log .

James Dempsey for the PCGEN Team -- 6 August, 2014.

Installation

Prerequisites

Java 6 or Java 7. PCGEN 6.3.4 is not compatible with Java 1.5.x and it is NOT SUPPORTED. To get the latest version of Java, go to java.com. PCGen has been tested with both Java 6 (including OpenJDK) and Java 7.

Instructions

Please see the Installation Documentation for information about installing PCGen.

Read the doc!

Please make sure you read the docs to familiarize yourself with the changes. You can access it from docs/index.html located in the PCGen directory.

Download PCGen

  • pcgen60304_full.zip:
    everything including GMGen and PDF
  • pcgen60304_win_install.exe:
    the full package bundle in an easy to use installer for Windows
  • pcgen60304_mac_install.dmg:
    the full package bundle in an easy to use installer for Mac (distributed as a disk image)

Custom Data Sets

If you have custom data created using either the old LST Editor, or the my_dataset example files, it is very important that you read the Upgrading PCGen and Upgrading home brews before installing your new version of PCGen.

Want to Help?

PCGen is an open source project. Anyone who wants to contribute whether it is Java programming, data set programming, documentation, helping new users, general cat herding, massage, walking the dogs, etc. is very welcome.

If you want to help drop a mail to PR@pcgen.org or Content@pcgen.org, or simply post a message with the title "I Want To Help" in one of the mailing lists.
