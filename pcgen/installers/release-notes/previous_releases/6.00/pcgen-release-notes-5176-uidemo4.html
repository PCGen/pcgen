What's new for UI Demo 4

This is our final demo of the new user interface. We've taken on board much of the excellent feedback from the beta group. The remaining suggestions will be incorporated as we merge in with the main program.

So please give this new version a try and continue to provide feedback on how the user interface is working.

Here's the main new features:

  • Improved consistency between tabs
  • Double click will now add/remove an item
  • Todo links take you to the relevant field
  • Add the Install Data function back in

James Dempsey for the PCGEN Team -- 31 January, 2012.
